# Website Traffic Analysis Dashboard

## Objective
Analyze website traffic patterns and user engagement using Power BI.

## Tools Used
- Power BI
- Excel

## KPIs
- Total Users
- Total Sessions
- First Time Visitors
- Returning Visitors

## Analysis Performed
- Website Traffic Trend
- Page Loads Trend
- Traffic by Day
- New vs Returning Visitors
- Visitor Distribution
- Traffic Growth Analysis

## Dashboard Features
- Interactive Filters
- Date Slicer
- Day Slicer
- Day of Week Slicer

## Key Insights
- Returning visitors contribute significantly to traffic.
- Website visits fluctuate across different days.
- Traffic growth trends can be monitored over time.
- User engagement remains stable across the analyzed period.
